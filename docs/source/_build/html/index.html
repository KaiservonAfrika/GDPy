GDPy Documentation¶

GDPy stands for Generating Deep Potential with Python, including a set of tools and Python modules to automate the structure exploration and the model training for machine learning interatomic potential (MLIP). It is developed and maintained by Jiayan Xu under supervision of Prof. P. Hu at Queen’s University Belfast.

Supported Potentials¶

eann, deepmd, lasp, nequip / allegro

Supported Expeditions¶

molecular dynamics, genetic algorithm, grand canonical monte carlo, graph-theory adsorbate configuration, artificial force induced reaction
